# HexHavoc client

Node.js 20+ and pnpm is required.

## Install pnpm

```bash
npm install -g pnpm
```

## To install dependencies:

```bash
pnpm install
```

## Running development server:

```bash
pnpm dev
```

## Bundling for production
```bash
pnpm build
```

> It will output HTML, JS and bundled assets in `dist/` directory
